# 睡眠页面五个子页面信息图重构 - Product Requirement Document

## Overview
- **Summary**: 根据五张截图（睡眠总时长、睡眠心率、睡眠血氧、睡眠呼吸率、睡眠心率变异性）重构睡眠页面日视图的五个子页面信息图，确保视觉细节与截图完全一致。
- **Purpose**: 使睡眠页面的数据可视化效果与原厂健康软件（vivo健康风格）保持一致，提升用户体验和专业感。
- **Target Users**: 睡眠页面的日视图用户

## Goals
- 重构睡眠总时长子页面的睡眠阶段分布图，与截图5完全一致
- 重构睡眠心率子页面的曲线图，与截图4完全一致
- 重构睡眠血氧子页面的曲线图，与截图2完全一致
- 重构睡眠呼吸率子页面的曲线图，与截图3完全一致
- 重构睡眠心率变异性子页面的曲线图，与截图1完全一致
- 统一五个子页面的头部布局（左对齐，主指标+辅指标结构）
- 确保所有图表细节（颜色、刻度、网格线、数据形状）与截图一致

## Non-Goals (Out of Scope)
- 不修改周视图和月视图
- 不修改底部导航和页面框架
- 不修改睡眠分数卡片以下的内容（分段睡眠、更多分析等）
- 不修改数据模型和业务逻辑

## Background & Context
- 当前睡眠页面日视图已有五个子页面的切换功能，但图表细节与参考截图存在差异
- 项目已确定使用蓝紫色系配色方案（#5B5CFF 为主色）
- 已有睡眠阶段分布、曲线图等基础组件，需要调整细节
- 参考 vivo 健康的 UI 设计风格

## Functional Requirements

### FR-1: 统一头部布局重构
- 将当前居中对齐的头部改为左对齐布局
- 左侧显示：指标名称（上方小字）+ 数值范围（大号数字+单位）+ "昨天"（底部小字）
- 右侧（可选）：辅助指标名称 + 数值（如平均血氧、基准心率、平均HRV）
- 呼吸率和总时长页面没有右侧辅助指标

### FR-2: 睡眠总时长子页面（睡眠阶段分布图）
- 显示完整的睡眠阶段分布图表（展开状态）
- Y轴标签从上到下：清醒、快速眼动、浅睡、深睡
- X轴标签：03:58、12:05、14:46
- 睡眠阶段条使用绝对定位，保持各阶段间距
- 右侧有蓝紫色渐变小睡条
- 网格线为虚线样式
- 无曲线图区域

### FR-3: 睡眠心率子页面
- 上方：压缩的睡眠阶段迷你条（垂直居中，带时间坐标）
- 下方：睡眠心率曲线图
  - 红色折线（#FF6B6B）
  - 红色渐变填充（上深下浅）
  - Y轴刻度（右侧）：40、80、120
  - X轴标签：03:58、14:46
  - 网格线为虚线
  - 右侧显示"睡眠基准心率 57次/分"辅助指标

### FR-4: 睡眠血氧子页面
- 上方：压缩的睡眠阶段迷你条
- 下方：睡眠血氧曲线图
  - 红色折线
  - 红色渐变填充
  - Y轴刻度（右侧）：90、95、100
  - X轴标签：03:58、14:46
  - 网格线为虚线
  - 右侧显示"睡眠平均血氧 97%"辅助指标

### FR-5: 睡眠呼吸率子页面
- 上方：压缩的睡眠阶段迷你条
- 下方：睡眠呼吸率曲线图
  - 红色折线
  - 红色渐变填充
  - Y轴刻度（右侧）：10、20、30
  - X轴标签：03:58、14:46
  - 网格线为虚线
  - 无右侧辅助指标

### FR-6: 睡眠心率变异性子页面
- 上方：压缩的睡眠阶段迷你条
- 下方：睡眠HRV曲线图
  - 红色折线
  - 红色渐变填充
  - Y轴刻度（右侧）：0、40、80、120
  - X轴标签：03:58、14:46
  - 网格线为虚线
  - 右侧显示"平均睡眠心率变异性 52毫秒"辅助指标

### FR-7: 底部指标滚动卡片
- 保持横向滚动交互
- 选中卡片为浅蓝色背景
- 确保五个指标卡片的数值与截图一致
- 卡片顺序：睡眠总时长 → 睡眠心率 → 睡眠呼吸率 → 睡眠血氧 → 睡眠心率变异性（以截图实际顺序为准）

## Non-Functional Requirements

### NFR-1: 视觉精确度
- 所有颜色、字体大小、间距必须与截图一致
- 图表曲线形状需与截图高度相似（通过调整模拟数据实现）

### NFR-2: 交互流畅性
- 子页面切换动画保持流畅
- 压缩/展开动画自然

### NFR-3: 响应式
- 在不同屏幕尺寸下保持布局一致性

## Constraints
- **技术**: React + TypeScript + CSS，使用现有技术栈
- **设计**: 必须严格遵循截图的视觉设计
- **依赖**: 基于现有 SleepPage 组件进行修改

## Assumptions
- 使用模拟数据来匹配截图中的曲线形状
- 底部指标卡片顺序以截图中可见的顺序为准
- 周视图和月视图不在本次修改范围内

## Acceptance Criteria

### AC-1: 头部布局与截图一致
- **Given**: 用户在睡眠页面日视图
- **When**: 查看任意子页面的头部区域
- **Then**: 头部为左对齐布局，左侧显示指标名称+数值+昨天，右侧（如有）显示辅助指标，字号和间距与截图一致
- **Verification**: `human-judgment`

### AC-2: 睡眠总时长图表与截图一致
- **Given**: 用户在睡眠总时长子页面
- **When**: 查看睡眠阶段分布图
- **Then**: Y轴顺序为清醒→快速眼动→浅睡→深睡，X轴标签为03:58/12:05/14:46，睡眠阶段条位置正确，小睡渐变条在右侧，网格线为虚线
- **Verification**: `human-judgment`

### AC-3: 睡眠心率图表与截图一致
- **Given**: 用户在睡眠心率子页面
- **When**: 查看曲线图
- **Then**: 上方有迷你睡眠阶段条，下方红色曲线图Y轴刻度为40/80/120，右侧显示基准心率57次/分，曲线形状与截图相似
- **Verification**: `human-judgment`

### AC-4: 睡眠血氧图表与截图一致
- **Given**: 用户在睡眠血氧子页面
- **When**: 查看曲线图
- **Then**: 上方有迷你睡眠阶段条，下方红色曲线图Y轴刻度为90/95/100，右侧显示平均血氧97%，曲线形状与截图相似
- **Verification**: `human-judgment`

### AC-5: 睡眠呼吸率图表与截图一致
- **Given**: 用户在睡眠呼吸率子页面
- **When**: 查看曲线图
- **Then**: 上方有迷你睡眠阶段条，下方红色曲线图Y轴刻度为10/20/30，无右侧辅助指标，曲线形状与截图相似
- **Verification**: `human-judgment`

### AC-6: 睡眠HRV图表与截图一致
- **Given**: 用户在睡眠心率变异性子页面
- **When**: 查看曲线图
- **Then**: 上方有迷你睡眠阶段条，下方红色曲线图Y轴刻度为0/40/80/120，右侧显示平均HRV 52毫秒，曲线形状与截图相似
- **Verification**: `human-judgment`

### AC-7: 底部指标卡片与截图一致
- **Given**: 用户在任意子页面
- **When**: 查看底部指标滚动卡片
- **Then**: 卡片数值正确，选中状态为浅蓝色背景，顺序正确
- **Verification**: `human-judgment`

### AC-8: 迷你睡眠阶段条垂直居中
- **Given**: 用户在非总时长的子页面
- **When**: 查看迷你睡眠阶段条
- **Then**: 迷你阶段条在其模块内垂直居中，带时间坐标标签
- **Verification**: `human-judgment`

### AC-9: 子页面切换动画流畅
- **Given**: 用户点击底部不同指标卡片
- **When**: 切换子页面
- **Then**: 展开/收起动画流畅自然
- **Verification**: `human-judgment`

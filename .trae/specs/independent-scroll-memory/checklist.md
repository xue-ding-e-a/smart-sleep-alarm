# 页面独立滚动与位置记忆 - Verification Checklist

## 功能验证

- [x] 首页滚动位置记忆：在首页向下滚动后切换到睡眠页，再切回首页，首页保持之前的滚动位置
- [x] 睡眠页滚动位置记忆：在睡眠页向下滚动后切换到我的页，再切回睡眠页，睡眠页保持之前的滚动位置
- [x] 我的页滚动位置记忆：在我的页向下滚动后切换到首页，再切回我的页，我的页保持之前的滚动位置
- [x] 三个页面独立滚动：三个页面分别滚动到不同位置（首页200px、睡眠页500px、我的页800px），互相切换后各自保持自己的位置，互不干扰
- [x] 首次进入从顶部开始：刷新页面后首次进入任意 Tab 页面，页面从顶部开始显示

## 动画与体验验证

- [x] 页面切换动画：切换页面时有淡入上移动画，效果与改动前一致
- [x] 滚动流畅度：三个页面滚动时保持流畅，无卡顿、掉帧现象（scroll 事件使用 passive 模式）
- [x] 切换无跳动：页面切换过程中无明显跳动、闪烁或白屏现象（使用 useLayoutEffect 在绘制前恢复位置）
- [x] 快速切换稳定性：快速在三个页面间来回切换，滚动位置始终正确

## 边界情况验证

- [x] 滚动到底部后切换：将页面滚动到底部，切换到其他页面再切回，仍保持在底部
- [x] 滚动到顶部后切换：将页面滚动到顶部，切换到其他页面再切回，仍保持在顶部
- [x] 轻微滚动后切换：只滚动一点点距离（如 10-20px）后切换，位置记忆正确
- [x] 长页面滚动：睡眠页内容较长，滚动到下方后切换再回来，位置正确

## 兼容性验证

- [x] 夜间主题适配：在夜间主题下滚动位置记忆功能正常工作
- [x] 不影响其他功能：页面上的所有按钮、开关、输入框等交互功能正常
- [x] TypeScript 编译通过：`npx tsc --noEmit` 无错误

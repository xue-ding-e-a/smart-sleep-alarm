# 睡眠页指标卡片拖拽滑动 - Verification Checklist

## 功能验证

- [x] 鼠标拖拽滑动：按下鼠标左键并向右拖动，卡片列表跟随移动（代码已实现）
- [x] 向左拖拽滑动：按下鼠标左键并向左拖动，卡片列表跟随移动（代码已实现）
- [x] 左边界限制：拖动到最左端时，卡片列表不再继续向左移动（代码已实现：Math.max(0, ...)）
- [x] 右边界限制：拖动到最右端时，卡片列表不再继续向右移动（代码已实现：Math.min(..., maxScrollLeft)）
- [x] 点击选中卡片：快速点击卡片（不拖动），卡片被选中并高亮（代码已实现：hasDraggedRef 区分）
- [x] 触摸滑动：触摸滑动功能保持正常工作（原生 overflow-x: auto 保留）

## 样式验证

- [x] 拖拽时鼠标样式：鼠标按下并拖动时，鼠标变为 grabbing 样式（代码已实现）
- [x] 拖拽时光标：鼠标悬停在卡片上时显示 grab 样式（CSS 已添加 cursor: grab）
- [x] 防止选中文本：拖拽过程中无法选中文本（代码已实现：userSelect: none）

## 边界情况验证

- [x] 快速拖拽：快速左右拖拽，响应流畅无卡顿（使用 requestAnimationFrame 风格的直接操作）
- [x] 拖拽到边界反弹：到达边界后再拖拽不会继续超出（代码已实现）

# 睡眠页面仿 vivo 健康风格重设计 - Product Requirement Document

## Overview
- **Summary**: 将 AI 睡眠唤醒闹钟应用的整体风格从暖白色改为仿 vivo 健康的蓝紫色系清新风格。重点重写睡眠页面的日/周/月三种视图，使其布局、数据展示、视觉风格与参考截图高度一致。其他页面（首页、个人页等）同步适配新的视觉风格。
- **Purpose**: 提供更专业、更接近主流健康应用的视觉体验，提升产品质感和用户认可度。
- **Target Users**: 使用睡眠监测智能硬件的健康关注者。

## Goals
- 整体配色方案改为蓝紫色系（主色 #5B5CFF / 类似 vivo 健康蓝）
- 睡眠页面日视图：睡眠阶段柱状图 + 横向滚动指标卡 + 睡眠评分 + 多段睡眠 + 展开/收起详情
- 睡眠页面周视图：周平均睡眠时长 + 周柱状图 + 8个指标卡片（各带迷你图表）
- 睡眠页面月视图：月平均睡眠时长 + 月柱状图 + 8个指标卡片
- 首页和个人页风格同步适配
- 所有卡片采用圆角 + 浅灰色边框 + 白色背景的简洁设计

## Non-Goals (Out of Scope)
- 不改变应用的功能逻辑和数据结构
- 不增加新的业务功能
- 不涉及后端 API 变更
- 不修改注册/登录页的核心交互

## Background & Context
- 当前应用使用暖金色主题，用户希望改为参考截图中的蓝紫色清新风格
- 参考截图为 vivo 健康应用的睡眠模块，设计简洁专业
- 睡眠页面是本次改造的重点，需要与截图数据和布局高度一致
- 技术栈：React + TypeScript + Vite + CSS Modules

## Functional Requirements
- **FR-1**: 主题系统改为蓝紫色系，包括主色、辅助色、背景色、文字色等 CSS 变量
- **FR-2**: 睡眠页面日视图重设计，包含：睡眠阶段柱状图、横向滚动的指标选择器（睡眠总时长/心率/血氧/呼吸率/心率变异性）、睡眠评分卡片、第1段睡眠（环形图）、更多分析展开/收起、第2段睡眠（小睡）、底部说明文字
- **FR-3**: 睡眠页面周视图重设计，包含：周平均睡眠时长、周睡眠柱状图、8个指标卡片（平均睡眠分数/平均卧床时长/平均深睡时长/睡眠心率/平均睡眠血氧/平均入睡时长/平均打鼾时长/平均不规律打鼾时长），每个卡片带迷你图表
- **FR-4**: 睡眠页面月视图重设计，包含：月平均睡眠时长、月睡眠柱状图、同周视图的8个指标卡片
- **FR-5**: 首页风格适配新主题
- **FR-6**: 个人页风格适配新主题
- **FR-7**: 底部导航、按钮、开关等 UI 组件同步适配

## Non-Functional Requirements
- **NFR-1**: 视觉一致性：所有页面的配色、圆角、间距、字体层级保持统一
- **NFR-2**: 响应式：在移动端和桌面端均有良好表现
- **NFR-3**: 性能：页面加载和切换无明显卡顿
- **NFR-4**: 可维护性：使用 CSS 变量管理主题，便于后续调整

## Constraints
- **技术**: React + TypeScript + Vite，使用原生 CSS，不引入额外 UI 库
- **业务**: 保持现有功能不变，仅做视觉和布局调整
- **数据**: 使用 mock 数据，数据值需与截图中保持一致

## Assumptions
- 参考截图中的主色调为蓝紫色（#5B5CFF 左右）
- 卡片圆角约为 16-20px
- 整体风格简洁、留白充足、层级清晰
- 睡眠页面是用户最关注的页面，优先级最高

## Acceptance Criteria

### AC-1: 主题配色系统
- **Given**: 用户打开应用任意页面
- **When**: 页面加载完成
- **Then**: 整体配色为蓝紫色系，与参考截图风格一致；背景为浅灰白，卡片为白色带浅灰边框，主按钮和强调色为蓝紫色
- **Verification**: `human-judgment`
- **Notes**: 对比截图，整体色调感觉一致即可

### AC-2: 睡眠日视图布局
- **Given**: 用户进入睡眠页面日视图
- **When**: 页面渲染完成
- **Then**: 顶部有日/周/月切换；显示睡眠阶段柱状图（深睡/浅睡/快速眼动/清醒）；下方有横向滚动的指标选择卡；睡眠评分卡片87分；第1段睡眠带环形图；"更多分析"可展开收起；第2段小睡；底部说明文字
- **Verification**: `human-judgment`

### AC-3: 睡眠日视图数据准确性
- **Given**: 用户查看睡眠日视图
- **When**: 对比参考截图
- **Then**: 睡眠总时长9小时18分(03:58-12:05)；睡眠心率49-93次/分基准57；睡眠血氧93-99%平均97%；睡眠呼吸率10-23.5次/分；睡眠心率变异性21-87毫秒平均52；评分87分睡眠良好；深睡1小时19分/浅睡5小时20分/REM1小时28分；第2段小睡1小时11分(13:35-14:46)
- **Verification**: `human-judgment`
- **Notes**: 数值需与截图完全一致

### AC-4: 睡眠周视图布局
- **Given**: 用户切换到周视图
- **When**: 页面渲染完成
- **Then**: 显示平均睡眠时长7小时47分钟；日期范围2026年6月20日至26日；周柱状图（周六~周五）；下方8个指标卡片（2列布局），每个带迷你图表
- **Verification**: `human-judgment`

### AC-5: 睡眠周视图数据准确性
- **Given**: 用户查看睡眠周视图
- **When**: 对比参考截图
- **Then**: 平均睡眠分数83分；平均卧床时长7小时38分钟；平均深睡时长1小时15分钟；睡眠心率45-109次/分；平均睡眠血氧97%；平均入睡时长无数据；平均打鼾时长无数据；平均不规律打鼾时长无数据
- **Verification**: `human-judgment`

### AC-6: 睡眠月视图布局

- **Then**: 显示平均睡眠时长7小时41分钟；日期范围2026年5月27日至6月26日；月柱状图（日期从125日到6月22日）；下方8个指标卡片（同月视图）

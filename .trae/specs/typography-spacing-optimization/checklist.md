# 字体和间距优化 - Verification Checklist

- [x] 字体栈使用现代无衬线中文字体，接近 vivo 系统字体风格
- [x] 大数字（评分、时长）字重够粗（700-900），视觉冲击力强
- [x] 字体大小层级清晰：超大数字 > 大标题 > 标题 > 正文 > 辅助文字
- [x] 字重层级明确：Black/Bold用于大数字，Semibold用于标题，Regular用于正文
- [x] 卡片内边距充足（20-24px），不拥挤
- [x] 模块之间间距合理，呼吸感好
- [x] 正文行高舒适（1.5-1.6），大数字行高紧凑（1.1-1.2）
- [x] 睡眠页面评分数字醒目，是视觉焦点
- [x] 睡眠时长数字粗体醒目
- [x] 卡片内文字层级清晰（标题/数值/辅助文字）
- [x] 环形图旁的文字排版整齐
- [x] 指标卡片文字大小适中
- [x] "更多分析"按钮文字清晰
- [x] 展开后统计卡片字体层级正确
- [x] 周/月视图标题和数值字体协调
- [x] 迷你图表下方日期标签字体合适
- [x] 首页字体风格与睡眠页一致
- [x] 个人页字体风格与睡眠页一致
- [x] 数字使用等宽（tabular-nums）
- [x] 整体排版与参考截图风格相似
